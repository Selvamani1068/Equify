    Intelligent Routing

    Intelligent Routing sits in the navigation menu directly alongside Traffic Management, under Routing Setup. At the time this guide was written, it is not yet available for configuration — opening it displays a dedicated Coming Soon screen rather than a working feature.

    What the application currently shows

    Selecting Routing Setup › Intelligent Routing opens a page headed "Something brilliant awaits," with the subtitle "We're crafting something extraordinary. Stay tuned for what's coming next." The page confirms the feature is "In active development" and displays three further pieces of information:

    Section What it shows
    Timeline A target release quarter (shown as Q4)
    Progress A completion percentage (shown at 85% at the time of writing)
    Get Ready A Notify Me link, allowing interested administrators to register for an update once the feature ships

    What to expect

    While the feature itself is not yet documented, its name and placement strongly suggest it will extend the rule-based strategies in Traffic Management with more dynamic, performance-aware routing — for example, automatically favoring whichever configured service provider is currently showing the best delivery rate or lowest latency on the Dashboard, rather than relying solely on rules an administrator has set manually. This is a reasonable inference based on the feature's name and its progress indicator, not a confirmed specification, and this chapter will be rewritten with full configuration steps once the feature is released and reviewed.

    In the meantime

    Until Intelligent Routing ships, build your routing policy using the seven proven strategies in Traffic Management, combined through Routing Combinations as needed. Administrators who want to be notified the moment Intelligent Routing becomes available can select Notify Me on the Coming Soon screen itself.
