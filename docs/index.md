# Equify Product Guide

Welcome to the official Equify user documentation.


## What is Equify?

Equify provides a centralized platform for managing communication traffic generated by enterprise applications and business systems. The platform enables organizations to integrate with multiple messaging providers, apply routing policies, monitor delivery performance, and track communication activity from a single interface.

Supported communication channels currently include:

- SMS
- WhatsApp

Additional channels can be integrated as business requirements evolve.
