# Edit a template

---

Use this procedure to update an existing WhatsApp template configuration. Only the **Service Type** and **Template Description** can be modified.

---

## Procedure

1. In the left navigation panel, select **Template Management**.
2. Select the required service provider from the **Service Provider** dropdown.

3. Locate the template that you want to edit.

4. In the **Actions** column, click the **Edit** icon.

    The **Edit Template** screen is displayed.


5. In **Service Type**, select the required category:
    - **Authentication**
    - **Utility**
    - **Marketing**

6. In **Template Description**, update the description as required.

7. Click **Save**.


The template is updated with the new configuration.

!!! Note
    Changes may take up to one minute to propagate across the system.
