# Add a template

---

Use this procedure to register a provider-approved WhatsApp template in Equify. Registered templates are used during message processing for validation, routing, and reporting.

---

## Before you begin

- Ensure the WhatsApp template is approved by the service provider.
- Identify the **template ID**, **template name**, and **category** assigned by the provider.
- Ensure the template ID is unique across the platform.
- Ensure the correct service provider and service type are known.

---

## Procedure

1. In the left navigation panel, select **Template Management**.
2. Select the service provider from the **Service Provider** dropdown.
3. Click **Add Template**.

    The **Add New Template** screen is displayed.

     ![Add Template](../../assets/images/add_template.png){ width="500" }

4. In **Service Type**, select the appropriate template category:
 
    Specifies the WhatsApp template category assigned by the provider.

    - **Authentication** – Used for user verification and authentication messages.  
    - **Utility** – Used for transactional and service-related notifications.  
    - **Marketing** – Used for promotional and engagement messages.

5. In **Template ID**, enter the provider-approved template identifier.

6. In **Template Name**, enter the template name as defined by the provider.

7. (Optional) In **Template Description**, enter additional details about the template.

8. Click **Add Template**.


The template is created and becomes available in template management.

!!! Note
    Changes may take up to one minute to propagate across the system.

---
