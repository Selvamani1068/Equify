# Delete a template

---

Use this procedure to remove an existing WhatsApp template from the system. Deleting a template permanently removes it from Equify and prevents it from being used in message processing.

---

## Procedure

1. In the left navigation panel, select **Template Management**.
2. Select the required service provider from the **Service Provider** dropdown.

3. Locate the template that you want to delete.

4. In the **Actions** column, click the **Delete** icon.

    The **Delete Template** dialog box is displayed.

5. Click **Delete**.


The template is deleted successfully.
