# Template Management Overview {#template_management}

---

The **Template Management** feature enables users to register and manage provider-approved WhatsApp templates used for business communications.

When a WhatsApp template is added in Equify, the platform captures the template information, associated service provider, and the template category assigned by the provider. This information is used during message processing to validate template usage and support operational reporting.

Users can use the **Template Management** feature to add approved templates, manage template information, track template categories, and maintain the template configurations required for WhatsApp communication.

---

## Opening Template Management

Select **Template Management** in the left navigation panel. It expands to display the available template-related functions.

---

## Template management components

| Section | Description |
|---|---|
| **Add Template** | Add provider-approved WhatsApp templates and associate them with a service provider. |
| **Template Management** | Edit or delete added WhatsApp templates. |
| **Template Details** | View template information and category assignments. |

---

## Template categories

Meta classifies WhatsApp template messages into the following categories:

| Category | Description |
|---|---|
| **Authentication** | Used for user verification and authentication-related communications. |
| **Utility** | Used for transactional, service, and account-related notifications. |
| **Marketing** | Used for promotional messages, campaigns, and customer engagement communications. |

When a template is registered, Equify captures the assigned category and associates it with the template configuration.

---

## Template validation and tracking

Equify uses registered WhatsApp templates to validate message requests and classify communication traffic.

During message processing:

- The submitted template is validated against the selected service provider.
- The template category is identified and associated with the message transaction.
- Category information is recorded for analytics, reporting, usage monitoring, and billing.
- Requests that reference unregistered templates are rejected before delivery.

This approach helps ensure provider compliance while maintaining accurate operational and reporting data.
