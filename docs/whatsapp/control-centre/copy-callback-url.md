# Copy Callback URLs

---

This guide describes the procedure for viewing and copying the callback URLs.

!!! note

    Callback URLs are managed by Equify and are read-only. To configure these URLs in the service provider portal, contact the Equify Support team.

---

## To copy callback URLs

1. Navigate to **Control Centre > Service Provider (SP) > SP Management**.

2. Click the **Actions** menu (⋮) for the service provider registration whose callback URLs you want to copy.

    ![update DB](../../assets/images/sp_view.png)

3. Select **Copy Callback URLs** from the **ACTIONS** menu.

The **Callback URLs** screens opens and displays the callback URLs for all configured response types.

!!! note
    You can copy individual callback URLs or copy all callback URLs as a JSON object for provider configuration.
